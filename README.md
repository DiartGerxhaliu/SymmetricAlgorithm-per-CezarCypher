# SymmetricAlgorithm-per-CezarCypher
mplementim i kodit të Cezarit si algoritëm simetrik, që mundëson enkriptim dhe dekriptim duke përdorur të njëjtin çelës (shift). Programi transformon vetëm shkronjat A–Z dhe a–z, ndërsa ruan të pandryshuara numrat, hapësirat dhe simbolet, duke garantuar përpunim të saktë të teksteve të përziera.
